=== Description ===
Simple plugin to display the currently PHP-MYSQL version at the end of "At a Glance" admin dashboard widget

=== Screenshots ===
Display the currently PHP-MYSQL version at the end of "At a Glance" admin dashboard widget

=== Installation ===
how to install the plugin 
1. From your WordPress dashboard
2. Search for 'PHP Version Display'
3. Activate 'PHP Version Display' from your Plugins page
